服务器基础知识之BIOS

发表于： 2024-09-08 分类于： Server
字数：114 阅读：≈ 1分钟 浏览：

BIOS （基本输入/输出系统）是计算机的微处理器在开机后用于启动计算机系统的程序。它还管理计算机操作系统 （OS） 和连接设备（如硬盘、视频适配器、键盘、鼠标和打印机）之间的数据流。

BIOS 的历史

BIOS 一词由美国计算机科学家 Gary Kildall 于 1975 年首次创造。它于 1981 年被整合到 IBM 的第一台个人计算机中，并在接下来的几年中在其他 PC 中流行起来，在一段时间内成为计算机不可或缺的一部分。然而，BIOS 的受欢迎程度已经减弱，取而代之的是一种更新的技术：统一可扩展固件接口 （UEFI）。英特尔在 2017 年宣布了一项计划，到 2020 年停止对传统 BIOS 系统的支持，取而代之的是 UEFI。

BIOS的使用

BIOS 的主要用途是充当操作系统和它们运行的硬件之间的中间人。从理论上讲，BIOS 始终是微处理器和 I/O 设备控制信息和数据流之间的中介。尽管在某些情况下，BIOS 可以安排数据从需要更快的数据流才能有效的设备（例如视频卡）直接流向内存。

BIOS 是如何工作的？

BIOS服务器的核心功能之一，作为主板芯片上的固件。BIOS 是一个程序，当 BIOS 启动计算机时，BIOS 首先执行一系列称为开机自检 （ POST ） 的检查，以确保服务器的各个硬件部分工作正常。在测试并确保启动设备正常运行后，BIOS 会将操作系统或其关键部分从硬盘或启动设备加载到计算机的随机存取内存中。

BIOS的4大功能

BIOS在服务器开机识别、配置、自检硬件并将其连接到操作系统。这些步骤的组合称为启动过程。

这些任务分别由 BIOS 的四个主要功能执行：

  • 开机自检 （POST）- 在服务器启动时，BIOS会对服务器硬件进行自检（POST，Power-On Self Test），检查CPU、内存、硬盘、键盘等设备是否正常工作。如果发现硬件问题，会发出警告声或停止启动。
  • 加载引导程序 - 自检完成后，BIOS会根据用户设置的启动顺序（Boot Order）寻找并加载存储设备上的引导加载程序（如UEFI引导程序），进而加载操作系统。
  • 提供设置界面 - BIOS提供了一个配置界面，允许用户调整系统设置，如日期和时间、启动顺序、硬盘模式、内存参数等。这个界面通常在启动时通过特定按键（如Del键或F2键）进入。
  • 互补金属氧化物半导体 （CMOS） 设置 - 这是一个配置程序，使用户能够更改硬件和系统设置。CMOS 是 BIOS 非易失性内存的名称。

四种主要类型的 BIOS

第一代 BIOS

第一代BIOS是原始版本，在计算机的早期使用。它存储在主板上的ROM芯片上。

EFI（可扩展固件接口）

EFI 旨在克服传统 BIOS 的一些限制，提供改进的优点，例如更好的可扩展性和可扩展性，以及更灵活的操作系统前环境。

UEFI（统一可扩展固件接口）

UEFI 是 BIOS 的现代版本，解决了其技术限制。它提供更快的启动时间、改进的安全措施，并支持大于 2.2 TB 的驱动器。

Mac 等效设备：开放式固件

对于 Mac 系统，等效于 BIOS 的功能称为开放固件。该系统基于 Open Firmware 标准，类似于 EFI。

如何进入BIOS

  • 启动计算机：首先启动或重新启动计算机。
  • 进入BIOS：计算机启动后，立即按相应的键进入启动屏幕。服务器厂商不同BIOS的进入方式也不一样。一些经常用作提示的键是 Del、Tab、Esc 和任何功能键 （F1-F12）
  • 配置BIOS：从启动屏幕中，选择进入 BIOS 设置的选项。用户可以更改硬件设置、管理内存设置、更改启动顺序或启动设备、重置 BIOS 密码以及其他配置任务。
  • 保存并退出：进行必要的更改后，保存更改并退出 BIOS。
